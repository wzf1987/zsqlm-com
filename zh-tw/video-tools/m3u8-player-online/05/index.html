下載了一個視頻文件，雙擊卻提示「不支援此格式」？別慌，這篇主流視頻格式大全一次帶你搞懂市面上所有常見的視頻播放格式，從此告別格式焦慮。

先搞懂一件事：格式 ≠ 編碼
#

很多人以為 MP4、AVI 就是視頻本身，其實不然。所謂視頻格式，本質上是一個「容器」，裡面裝的才是真正的視頻編碼（如 H.264、H.265）和音頻編碼（如 AAC、MP3）。同一個 .mp4 後綴，裡面可能塞了不同的編碼組合，這就是為什麼有時同一種格式，這台電腦能播、那台卻不行。

記住這個公式：視頻格式 = 容器（封裝）+ 視頻編碼 + 音頻編碼。搞清楚這三層，你就不會再被後綴名迷惑了。

六大主流視頻格式詳解
#

1. MP4格式——當之無愧的「萬能格式」
#

MP4（MPEG-4 Part 14）是目前兼容性最強的視頻格式，幾乎所有設備、平台、瀏覽器都能播。它採用 H.264 或 H.265（HEVC） 編碼，能在較小體積下保留接近 DVD 的畫質。無論是手機、電腦還是智能電視，MP4 都是首選。網絡分享、社交媒體、線上流媒體，九成以上都用它。

2. MKV格式——高清黨的最愛
#

MKV（Matroska）是一種開放源碼的多媒體容器，堪稱「萬能容器」。它能在一個文件中塞入多條音軌、多種字幕，甚至多個視頻流。國語、粵語、英語隨便切，簡體、繁體字幕自由選。正因如此，MKV 在高清電影收藏圈極受歡迎。缺點是部分移動設備和老舊播放器可能不支援。

3. AVI格式——老牌經典，但體積龐大
#

AVI（Audio Video Interleaved）由微軟於1992年推出，是最長壽的視頻格式之一。優點是畫質好、壓縮標準可任意選擇；缺點也很明顯：文件體積巨大，且壓縮標準不統一，經常出現高版本播放器打不開早期編碼視頻的情況。如果你還在用 AVI，建議盡早轉換。

4. MOV格式——蘋果生態的親兒子
#

MOV 由蘋果公司開發，默認播放器為 QuickTime。它支援高品質視頻和多軌道，甚至能保存 alpha 通道，是 Final Cut Pro、After Effects 等專業軟件的標配。iPhone 拍的視頻基本都是 MOV。缺點是 Windows 兼容性較弱，文件體積也偏大。

5. WMV格式——微軟的流媒體利器
#

WMV（Windows Media Video）是微軟專為網絡流媒體設計的格式，壓縮率高、體積小，適合在線邊下邊播。但在非 Windows 設備上兼容性較差，如今已逐漸被 MP4 取代。

6. FLV / F4V格式——短視頻時代的幕後英雄
#

FLV 曾是各大視頻網站的標配，文件極小、加載極快。而 F4V 作為升級版，支援 H.264 高清編碼，畫質更清晰。隨著 Flash 淘汰，FLV 已大幅衰落，但你在抖音、快手下載的視頻，往往還是這類格式。

格式不兼容怎麼辦？
#

最簡單的方案：用格式轉換工具將視頻轉為 MP4（H.264編碼），這是兼容性最廣的「安全牌」。也可以安裝萬能播放器如 VLC 或 PotPlayer，它們幾乎通吃所有格式，一個軟件搞定全部。

#

日常使用認準 MP4，高清收藏選 MKV，蘋果用戶看 MOV，其他格式了解即可。掌握這些視頻格式知識，無論是下載、剪輯還是投屏，你都能遊刃有餘。
